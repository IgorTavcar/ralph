## Codebase Patterns
- Python project uses `pyproject.toml` with setuptools backend
- CLI entry point: `audiochap.cli:main` registered as `audiochap` script
- Supported audio formats: .mp3, .wav, .m4a, .ogg, .flac (defined in `SUPPORTED_AUDIO_EXTENSIONS`)
- All errors go to stderr with exit code 1; argparse default exit code 2 is overridden via `_Parser` subclass
- `validate_args()` handles all post-parse validation and exits directly on failure
